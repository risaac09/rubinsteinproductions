# Rubinstein Productions

Facilitation and film for people in transition. A camera, a conversation, and a short film that says what you've become.

**Live site:** [rubinsteinproductions.com](https://rubinsteinproductions.com)

## Stack

- React + Vite
- GSAP scroll animations
- Deployed via Netlify

## Development

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
npx netlify-cli deploy --prod --dir=dist --site=8ed4687e-468f-4119-acc9-eb0ba077c6ce
```
